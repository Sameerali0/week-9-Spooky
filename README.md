# Ghost Catcher

Ghost Catcher is a fun and interactive game where your goal is to tap and catch as many ghosts as possible within 30 seconds. Each ghost caught earns you points, and the missed ghosts disappear after a short time. Your highest score is saved locally, so you can always try to beat your own record!. The game includes smooth animations, background music, and sound effects for a complete gaming experience.

## Technologies Used
- HTML
- CSS
- JavaScript
